# Civilization VII Mod ID Rules

Use stable IDs so mods stay easy to search, validate, deploy, and publish.

## Naming Pattern

- Folder: `better-town-growth`
- Mod ID: `erunyan6-better-town-growth`
- XML prefix: `ERU_BETTER_TOWN_GROWTH`
- Text key name: `LOC_ERU_BETTER_TOWN_GROWTH_NAME`
- Text key description: `LOC_ERU_BETTER_TOWN_GROWTH_DESCRIPTION`

## Rules

- Use lowercase kebab-case for folders and mod IDs.
- Prefix all personal mod IDs with `erunyan6-`.
- Prefix custom XML IDs with `ERU_`.
- Prefix localized text with `LOC_ERU_`.
- Never reuse a mod ID for a different mod concept.
- Keep one `.modinfo` file at the root of each mod folder.
- Keep one `manifest.json` file at the root of each mod folder.
